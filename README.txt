This is README file for smile_puzzle solver coded in C++ language.

To see result use terminal. Navigate to folder where main.cpp, Utils.cpp, Utils.h, Node.cpp,
Node.h and output.txt are located. Call output.exe.

To change input, open main.cpp and add some new grid and call solve_puzzle(new_grid) function.

Compile with: 
	g++ -o output.exe main.cpp Utils.cpp Node.cpp
